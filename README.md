# ✂️ Rock Paper Scissors — Multi-Class Image Classification using Keras

A deep learning project that classifies hand gesture images into three categories — **Rock**, **Paper**, or **Scissors** — using a Convolutional Neural Network (CNN) built with Keras.

---

## 📌 Description

This project tackles multi-class image classification using a CNN trained on the Rock-Paper-Scissors dataset. The model learns to distinguish between three hand gesture classes from image input, demonstrating core concepts of deep learning including data augmentation, CNN architecture design, and multi-class softmax classification.

## ⚙️ Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/priya-zha/paper_scissors_rock-multi-class-image-classification-using-keras.git
cd paper_scissors_rock-multi-class-image-classification-using-keras
```

### 2. Install Dependencies
```bash
pip install tensorflow keras numpy matplotlib jupyter
```

### 3. Launch the Notebook
```bash
jupyter notebook "paper_scissors_rock .ipynb"
```

---

## 🚀 Usage

1. Open the notebook in Jupyter or Google Colab
2. Run cells in order to:
   - Load and preprocess the Rock-Paper-Scissors dataset
   - Apply image data augmentation
   - Build and train the CNN model
   - Evaluate accuracy on test data
   - Visualize predictions
